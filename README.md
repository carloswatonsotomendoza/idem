Ejemplo de arquitectura limpia de tareas pendientes

Este repositorio contiene una aplicación sencilla de tareas pendientes creada con Node.js y React.

El objetivo de este proyecto es demostrar:
- Principios de arquitectura limpia
- Principios SOLID
- Prácticas TDD
- Código limpio y legible
- Configuración básica de CI

## Estructura del proyecto

- `/backend`: API de Node.js + TypeScript
- `/frontend`: Aplicación de React + TypeScript
